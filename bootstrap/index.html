HTML (HyperText Markup Language) är standard markup-språket för hemsidor. Det är inte ett programmeringsspråk, utan bestämmer strukturen av allt på din hemsida. För att kunna göra mer med en hemsida integrerar man ofta CSS och Javascript. HTML är standarden för alla hemsidor på internet och den senaste versionen av HTML är HTML5.

HTML-element är byggnadsblocken för din hemsida. Dessa definieras av tags som öppnas med <> och stängs med (alla HTML-element behöver dock inte en slut-tagg). Tags skapar möjligheter för att strukturera HTML-dokument och hjälper därmed me läsbarhet. Några exempel på HTML-element är bilder (<img>), rubriker (<h1> -<h6>) och länkar (<a>).

Ett HTML ID är en attribut som ett element kan ha som är unikt för detta element. Med hjälp av ID kan man ge egenskaper till detta element genom t.ex CSS.

Ett HTML klass är en attribut som flera element kan ha. Med hjälp av klasser kan man ge egenskaper till dessa element genom t.ex CSS. Ett exempel på en vanlig klass är container.

Metadata är information om information. I HTML används en tagg för att definiera metadata i HTML dokumentet. Detta kan t.ex vara vilka karaktärer som kan användas i dokumentet.

En responsiv hemsida är en hemsida som ser bra ut på alla enheter, vare sig det är en mobil, dator eller ipad. Responsivitet gör att hemsidan automatiskt justeras baserat på skärmens storlek och det är otroligt viktigt i dagens digitala samhälle.

HTTP står för Hypertext Transfer Protocol och möjliggör kommunikation på the World Wide Web mellan klienter och servrar. Detta görs genom att skicka HTTP Requests och få HTTP Responses.

HTTPS står för Hypertext Transfer Protocol Secure och är en kombination mellan HTTP och Transport Layer Security protocol. TLS är protokoll för autentisering och säkerhet som implementeras över hela webben.

JQuery är ett snabbt och komprimerat Javascript bibliotek med många funktioner som är tänkt att förenkla bl.a HTML- och CSS-modifikation.

Chromium är en webbläsare baserad på samma öppna källkod och med samma funktioner som Google Chrome, Microsoft Edge och Torch. Chromium innehåller däremot varken autouppdatering eller användningsspårning och den har en annan logotyp.

SVG står för Scalable Vector Graphics. Istället för att utgöras av pixlar som en png eller jpeg består SVG av 2D-vektorgrafik, vilket b.la innebär att man kan zooma in hur mycket som helst på bilden utan att den förlorar sin skärpa.

PNG är, likt JPEG, ett format för digitala bilder. PNG-24 (24 bit) stödjer fler färger än PNG-8 (8 bit) och kan ofta vara större i filstorlek än t.ex JPG. Till skillnad från JPEG förstörande komprimering använder PNG icke förstörande komprimering, vilket innebär att man kan få en bild som är mycket lik originalet.

DNS står för Domännamnssystemet. Det är ett system som används för att förenkla IP-adressering av datorer på IP-nätverk som internet. Domännamn och IP-adresser kopplas ihop, vilket är där den egentliga kommunikationen sker. DNS används också som en mer generell databas.

CDN står för Content Delivery Network, vilket på svenska blir innehållsleveransnätverk, vilket är ett nätverk av proxy-servrar utspridda geografiskt.

Komprimerad data är data som komprimerats, vilket betyder att den behöver färre bits för att representera datan och därmed tar den mindre plats.

CSS Flexible Box Layout är en CSS web layout modell som används för att positionera element på din hemsida och skapa en responsiv layout.

Grid är, likt Flexbox, ett sätt att skapa layouter med CSS på din hemsida. Grid använder rader och kolumner.

Code snippets är rader av återanvändbar kod som används för att förkorta processen av att koda.

Med WebAssembly (WASM) kan du skriva kod på t.ex c++, rust eller JavaScript och sedan konvertera det till ett binärt format, vilket gör att det kan köras på alla browsers.

Bootstrap är ett Javascript och CSS bibliotek som gör det mycket enklare och snabbare att bygga responsiva hemsidor. Att använda local Bootstrap innebär att ladda ned bootstrap på din dator och sedan använda det, till skillnad från att använde en länk som gjort i det här dokumentet.
